### Design Specifications:

**Background Image:**
- **Style:** Soft natural scene (Slovak mountains, forest, gentle sunrise/sunset)
- **Mood:** Peaceful, hopeful, reverent
- **Treatment:** Slight blur, warm color grading
- **Overlay:** Deep green (#2F3E35) at 40% opacity

**Typography:**
- **Main Headline:** 
  - Font: Cormorant Garamond Bold
  - Size: 56px
  - Color: White (#FFFFFF)
  - Letter-spacing: 2px
  - Text-align: Center
  
- **Subheadline:**
  - Font: Inter Regular
  - Size: 20px
  - Color: Warm Beige (#F2EDE4)
  - Text-align: Center
  - Max-width: 700px, centered

**CTA Buttons:**
- **Primary (left):**
  - Background: Soft Gold (#C7A87E)
  - Text: Deep Green (#2F3E35)
  - Size: 18px Inter Medium
  - Padding: 18px 32px
  - Border-radius: 8px
  - Box-shadow: 0 4px 12px rgba(0,0,0,0.15)
  - Hover: Lighten gold by 10%

- **Secondary (right):**
  - Background: Transparent
  - Border: 2px solid white
  - Text: White
  - Same sizing as primary
  - Hover: Fill white, text changes to deep green

**Emergency Phone:**
- **Display:** Always visible in top-right
- **Icon:** Phone icon in soft gold
- **Animation:** Subtle pulse effect (slow, dignified)
- **Mobile:** Converts to sticky floating button

## 8. Color Usage Summary

| Element | Color | Usage |
|---------|-------|-------|
| **Primary backgrounds** | Warm Beige (#F2EDE4) | Main sections, cards |
| **Accent backgrounds** | Deep Green (#2F3E35) | Footer, accent sections |
| **White sections** | White (#FFFFFF) | Alternating sections for rhythm |
| **Text primary** | Deep Green (#2F3E35) | Headings, body text |
| **Text secondary** | Silver Grey (#B4B7B1) | Captions, metadata |
| **CTA buttons** | Soft Gold (#C7A87E) | Primary actions |
| **Hover states** | Lightened Gold | Interactive feedback |
| **Borders/dividers** | Silver Grey (#B4B7B1) | Subtle separation |
